Pro メンバーシップ

Patreon や GitHub Sponsors で p5play を支援して、 Pro メンバーになりましょう。
p5play-pro

ドキュメントは近日公開予定！

商用利用

Pro メンバーは、 p5play プロフェッショナルライセンスが適用されることで、制作物のソースコードを公開することなく、 p5play を商用利用することができます。

iOS アプリテンプレート

Pro メンバーはテンプレートを使うことで、 Swift のコーディングをすることなく iOS アプリを制作できます。Apple の App Store でゲームを販売したり、広告収入を得たりすることが可能になります！

カリキュラム

「p5play ゲームデザイン基礎」は、 p5play v3 の開発者である Quinton Ashley が制作したカリキュラムです！

Pro メンバーは、このカリキュラムのスライドや実習内容へのアクセス権を得られるので、皆さまのゲームデザインスキルのレベルアップに、存分にお役立てください。

リソース

  • p5play-vscode : VSCode 拡張機能
  • p5play-template : スターターコード
  • p5play-desktop : デスクトップアプリ用テンプレート

教育機会の均等化（イクイタブル・アクセス）にむけて

もし Pro メンバーシップを購入する経済的余裕がない場合は、 info@p5play.org までお問い合わせください。その際、メールの件名に "Equitable Access" とご記入の上、 p5play の使用目的について、ご説明をお願いいたします。
