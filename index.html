:

p5play is a JavaScript game engine that uses p5.js for rendering and Box2D for simulating physics.

It was created by Computer Science teachers to help you turn your ideas into JavaScript code! Our goal is to make coding more accessible for artists, designers, educators, and beginners. Behind the scenes p5play v3 uses the planck.js physics simulator, a JS port of Box2D, which has been used in popular 2D games like Tiny Wings and Angry Birds.

Ready to learn how to develop your own games or interactive art? Start by reading the "Learn" reference pages!
